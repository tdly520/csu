WMI SQL Server™ Administration Provider

Release Notes

June, 2000

·        The WMI SQL Server™ Administration provider will return the error code WBEM_E_PROVIDER_FAILURE (0x80041004) when a failure occurs on an operation.  These errors are typically caused by a problem executing the requested operation within SQL Server™.  The error should not be interpreted as a failure within the provider itself.  A WMI extended status object is normally returned with this error and will provide more information about where the problem occurred.  See the WMI SQL Server™ Administration provider documentation section called "Error Reporting and Handling" for more details.

 

Please see the WMI SQL Server™ Administration provider documentation for more information about installation and use of the provider.

More information about Microsoft Windows Management Instrumentation (WMI) can be found at http://msdn.microsoft.com/downloads/sdks/wmi/default.asp.
